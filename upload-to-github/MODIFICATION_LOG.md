# 三千彩环保订购系统修改记录

## 2026-05-24

- 增加仓库订单的“导出Excel”按钮，可把单个订单输出为 Excel 可打开的送货单文件。
- 保留原来的“送货单”打印功能，方便后续继续做模板纸套打。
- 仓库页面现在会保留并显示“已发货”订单，方便仓库回看近期完成记录。
- 新增 `records.html` 订单记录查询页，用于查看全部历史订单，完成订单也能查到。
- 老板页、系统管理页、仓库页增加“订单记录”入口。
- 记录页保留“复制Excel数据”功能，方便把全部订单数据复制到 Excel 做汇总或备份。

## 2026-05-25

- 仓库订单改为每个产品单独填写实际发货公斤数，送货单和 Excel 按实际发货数输出。
- 仓库订单卡片去掉不需要的总重量/发货合计显示，减少重复信息。
- 增加仓库“库存管理”模块，可录入每个产品当前库存。
- 订单到仓库时，每个产品旁边显示实时库存。
- 仓库点击“库存够，正常发货”后，系统会按实际发货公斤数自动扣减库存。
- 增加一批本地测试库存数据，方便现场试用。
- 业务员页面增加本月销售额、本月销量、本月订单数和客户欠款统计。
- 新增财务页面，可登记客户收款，客户欠款会按收款金额减少。
- 销售助理产品管理增加商品类别，默认包含色浆、助剂、分散染料、其他。
- 销售助理可新增商品类别，新增产品时可选择类别，前台按商品类别筛选。
- 销售助理现在可以删除原有默认色浆；默认产品删除后会从前台、价格和库存中隐藏，并保留恢复入口。
- 产品管理拆成独立页面，并在销售助理页面增加“产品管理”跳转按钮。
- 客户管理拆成独立页面，并在销售助理页面右上角增加“客户管理”按钮。
- 业务员、销售助理、老板页面增加“最新已发货”通知模块。
- 业务员、销售助理、老板页面增加订单记录入口，历史订单统一到记录页查询。
- 增加 8 个业务员独立链接；通过对应业务员链接下的订单、销售统计、发货通知和订单记录会按业务员分开显示。
- 业务员下单页增加“选择下单公司”功能，选中公司后自动带出客户价格、联系人、地址和已沟通产品；业务员订单会直接进入销售助理处理流程。
- 销售总结升级为日/周/月报表，支持按公司和产品筛选；业务员只看自己名下订单，销售助理、老板、财务看全部订单。
- 新增独立销售总结页面，并在业务员、销售助理、老板、财务页面右上角增加“销售总结”按钮。
- 销售总结增加开始日期和结束日期，可按具体时间范围统计。
- 增加订单结算单价功能：保留下单单价，出货后可调整结算单价，销售总结、欠款和送货单优先按结算价计算。
- 订单结算价输入改为折叠显示，默认只显示“调整结算价”按钮，点击后再展开编辑。
- 新增入库网站和出库网站：入库可登记生产批号、产品和数量；出库时仓库可为订单产品选择批号，订单会保留批号记录并扣减批号剩余库存。
- 客户下单增加“跳过老板审核”选项：默认仍走老板审核，勾选后可直接进入业务员处理。
- 业务员、销售助理、老板、财务主页移除内嵌销售总结，仅保留右上角“销售总结”按钮进入独立报表页。
- 客户下单页新增“查看我的订单”，客户可查看订单状态和明细；已发货订单可提交退货申请，进入销售助理后台处理。
- 客户订单状态拆成独立 `customer-orders.html` 页面；下单页只保留“查看我的订单”跳转入口，减少右侧订单栏拥挤。
